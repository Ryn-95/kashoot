# Kashoot - Films de mariage haut de gamme

Site web premium pour l'agence de vidéaste mariage Kashoot.

## Stack technique

- Next.js 14 (App Router)
- React 18
- TypeScript
- TailwindCSS
- Framer Motion
- Lucide React (icons)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```

## Structure du projet

```
kashoot/
├── app/                  # Pages Next.js (App Router)
├── components/           # Composants React
│   ├── layout/          # Navigation, Footer
│   ├── sections/        # Sections de la page d'accueil
│   └── ui/              # Composants UI réutilisables
├── lib/                 # Utilitaires
└── public/              # Assets statiques
```
